## Overview

![screenshot](https://i.ibb.co/NnYXVNV/brows.png)

Static page developed to practice flexbox display. My focus is to try build simple projects using the best coding practices in order to enhance my frontend skills.

- Responsive units were used.
- Mobile first approach.
- The page must be responsive for tablets and desktop.
- BEM method of css architecture implemented.
